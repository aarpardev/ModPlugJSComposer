OpenMPT JS Tracker Music Maker
(Slim Version)

This is a quick fun little Javascript app that composes random music for OpenMPT Tracker. Simply enter some notes, select "Compose", scroll down, copy the resulting text starting from ModPlug Tracker XM, and paste it into a tracker channel in your project in OpenMPT. (Note: currently works for .xm projects at this point).

Check out the full source code on Github.
